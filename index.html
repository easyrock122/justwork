깃허브에 업로드하기

1)git init

2)git remote add origin "내 깃 헙 주소"

3)git add .

4)git commit -m"~~~~~~~"

5)git push origin master

%변경된 부분 업로드시 3~5번 반복

git에 쓰이는 명령어

-git branch 브랜치명(새로운 브랜치 생성)

-git checkout 브랜치명(해당 브랜치로 이동)

-git push origin 브랜치명(원격 저장소의 특정 브랜치에 프로젝트 저장

-git pull origin 브랜치명(원격 저장소의 특정브랜치에 변경사항 pull

git clone 원격저장소.git(원격 저장소의 파일 전체 복사)

git status(깃의 상황, staging area에 커믿되지 않은 것이 있는지 변경사항이 있는지, 어떤브랜치에 있는지 서술
